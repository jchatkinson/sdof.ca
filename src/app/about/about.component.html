A simple tool for studying the behavior of single-degree-of-freedom systems

Version 0.1 Created by Jeremy Atkinson

SDOF is a free tool to study the behavior of single-degree-of-freedom (SDOF) systems under various types of excitation. It can also be used to generate different excitation patterns and export them to a file.

Under the hood, the tool is using recurrence formulas to calculate the system response. The method is rather straightforward, and a detailed explanation is available in many structural analysis textbooks and online.

SDOF was built using Angular 8.0 and tailwindCSS, and is hosted on Github.

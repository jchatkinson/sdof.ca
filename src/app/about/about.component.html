A simple tool for studying the behavior of single-degree-of-freedom systems

Version 0.1 (still under development) Created by Jeremy Atkinson

SDOF is a free tool to study the behavior of single-degree-of-freedom systems (SDOFs) under various types of excitation. It can also be used to generate various excitation patterns, quickly inspect ground motion files, and export excitation patterns to a file.

Under the hood, the tool is using recurrence formulas to calculate the system response. The method is rather straightforward, and a detailed explanation is available in many structural analysis textbooks and online.

SDOF was built using Angular 8.0 and tailwindCSS, and is hosted on Github. There are still some bugs to be fixed and features to be added. If you see something wrong you can let me know at: jeremy@sdof.ca
